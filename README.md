## ğŸ¤– Introduction

**Prepwise** is a modern job interview preparation platform built with **Next.js** and **Firebase**, enhanced by **AI voice agents** and **LLM-powered feedback**.

The platform allows users to:

- Create AI-driven mock interviews
- Speak with voice-based interviewers
- Receive structured feedback and scoring
- Track and improve interview performance over time

The project focuses on **clean architecture**, **reusability**, and **real-world production patterns** using modern web technologies.
